# Limpopo Chess Academy Games Database

A Next.js application for viewing and analyzing chess games from the Limpopo Chess Academy database.

## Features

- Chess game viewer with interactive board
- Responsive design for desktop and mobile
- Game database browsing and filtering

## Installation

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Project Structure

- `src/app/view/page.tsx` - Main game viewer
- `public/engine/` - Engine files directory
