# Limpopo Chess Academy

Front-end web application for the Limpopo Chess Academy — coaching, tournaments, rankings and registrations in Limpopo, South Africa.

## Stack

- **Next.js 16** (App Router, Turbopack)
- **Supabase** (auth, database, storage)
- **Tailwind CSS v4**
- **TypeScript**
- **pnpm**

## Getting started

```bash
pnpm install
pnpm dev
```

The dev server runs on `http://localhost:8008`.

## Environment variables

Copy `.env.example` to `.env.local` and fill in your Supabase project URL and anon key.
